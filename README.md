# React To-Do List 

A dynamic **to-do list application** built with **React**, **Context API**, and **Local Storage**. Users can **add, edit, delete, and mark tasks as completed**, with all changes persisted across sessions. The project demonstrates **state management with Context**, **component-based architecture**, and **responsive UI design** using **Tailwind CSS**.

---

## Features

- Add new tasks  
- Edit existing tasks  
- Delete tasks  
- Mark tasks as completed  
- Persistent data with **Local Storage**  
- Responsive UI using **Tailwind CSS**

---

## Tech Stack

- **React**  
- **Context API** for state management  
- **Local Storage** for persistence  
- **Tailwind CSS** for styling  
- **JavaScript (ES6+)**
